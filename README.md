# vibeslayer profile

A dark personal profile page built with React, TypeScript, Vite, Tailwind CSS, Framer Motion, and YouTube playback.

## Getting started

```bash
pnpm install
pnpm dev
```

Open the local URL printed by Vite.

## Scripts

```bash
pnpm dev
pnpm build
pnpm preview
```

## Customize

- Edit profile text in `src/components/ProfileCard.tsx`.
- Edit social links in `src/data/socials.ts`.
- Edit music tracks in `src/data/tracks.ts`.
- Replace images in `assets/` or `src/assets/`.

## Build

```bash
pnpm build
```
